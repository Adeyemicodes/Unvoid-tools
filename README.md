# Patient Unvoid Tool

Secure tool for unvoiding patient records in OpenMRS database.

## Quick Start

1. Edit `unvoid_config.ini` with your database credentials
2. Run the application (double-click executable)
3. Login with password: pibtib
4. Enter patient identifier and follow on-screen instructions

## Configuration

Edit `unvoid_config.ini`:
```ini
[database]
host = localhost
user = openmrs_user
password = YOUR_PASSWORD_HERE
database = openmrs
```
